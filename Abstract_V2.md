Title:
An Autonomous Multi-Agent AI Platform for End-to-End Flood Disaster Response Using Fine-Tuned Vision–Language Models and Geospatial Analytics

Keywords: Multi-agent AI; Flood disaster response; Vision–language models; Geospatial analytics; Emergency decision-support

Abstract:

Flood disasters demand rapid, multi-dimensional situational awareness spanning hazard detection, infrastructure impact analysis, evacuation logistics, and community coordination, yet conventional GIS workflows address these needs in isolation and consume hours of expert labour, often at the cost of lives. This study presents the design, implementation, and evaluation of an autonomous multi-agent AI platform that delivers comprehensive end-to-end flood emergency response through a single user interaction, accessible to non-GIS-expert first responders, thereby transforming fragmented workflows into a unified, minutes-long autonomous operation. The system architecture comprises four specialised agents orchestrated through the CrewAI framework with a Llama 3.2 reasoning backbone, collectively performing fourteen integrated response functions. A Qwen2-VL-7B vision–language model, fine-tuned via Low-Rank Adaptation on the AMD Instinct MI300X GPU, performs satellite image damage classification and generates damage-zone GeoJSON polygons. A live-data agent retrieves hourly precipitation from the Open-Meteo API to generate flood polygons, which are spatially intersected with OpenStreetMap road networks using GeoPandas to identify blocked segments. A routing agent computes optimal hazard-avoidant evacuation paths via NetworkX shortest-path algorithms. Beyond core analytics, supplementary modules provide early warning alerts based on precipitation thresholds, seven-day predictive flood forecasting, emergency shelter and supply-point localisation via Overpass API queries, crowd-sourced geotagged incident reporting, missing-person tracking, emergency SOS dispatching, and an offline-resilient mapping capability that caches road graphs and exports self-contained HTML maps for connectivity-degraded environments. A FastAPI backend serves all services to a Streamlit interface featuring multi-layer Folium map visualisation. The deployed pipeline produced complete flood situational reports in under sixty seconds, with the fine-tuned vision model returning damage GeoJSON in under five seconds per image. The system reliably identified blocked road segments and generated topologically valid evacuation routes around dynamically computed flood zones. These findings demonstrate that coupling fine-tuned vision–language models with multi-agent orchestration, predictive analytics, and participatory community reporting offers a scalable, resilient template for AI-driven disaster decision-support in resource-constrained emergency settings.
